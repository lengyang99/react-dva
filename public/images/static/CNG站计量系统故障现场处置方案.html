事故特征

流量计不计量、管道异物、连接处外漏或传输存在问题可能引起计量失效、火灾爆炸等事故导致人员伤亡及财产损失

1.1 外漏：引起火灾，造成人身伤害、财产损失。

1.2 管道异物：造成无法计量，影响供气。

1.3 传输问题：影响自动加臭；不能随时监控。

2  现场处置措施

2.1 不能马上解决，立即启动备用管路（即备用流量计）。

2.2 流量计不计量：到现场观察流量计，如机械表不运转，可能有异物卡住涡轮，应打开流量计清理，如仍不能恢复正常，说明流量计机械部分故障；如发现自动屏幕无显示或显示信息异常，应通知相关人员查找原因，不能自行解决的问题，应与厂家联系等待处理。

2.3 外漏：关闭上下端阀门，放散后维修。

2.4 传输系统问题：无自控流量信号，首先检查插口是否松动，再到现场观察流量计，如流量计机械表运转正常，判断信号传输系统问题，让相关人员查电路是否正常，自己不能解决，报请有关部门处理。

3  现场注意事项

3.1 时刻关注远程控制信号，及时发现并处理计量系统问题。

3.2 根据实际瞬时流量，合理地选择流量计大小、数量，使其处于最佳运行状态。

3.3 定期检漏，发现接口处漏气要及时修复。
